CSS Hack

指一种特殊的代码，这种代码只能在某些浏览器中识别，其他浏览器不能识别。一般用来解决兼容性问题

条件hack只对IE浏览器有效，其他浏览器会将此识别为注释，IE10以上不支持这种方式

格式：

```HTML
<!- -[if IE x]><![endif]- ->		在IE*的版本里显示
<!- -[if lt IE x]><![endif]- ->		 在小于IE*的版本里显示（gt：大于、lte：小于等于）
```

属性hack专门解决样式兼容问题

`_属性`		 选择IE6及以下版本

`*属性`		 选择IE7及以下版本

`属性\9`		选择IE6以上

`属性\0`		选择IE8及以上和Opera15以下的浏览器
